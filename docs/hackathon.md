# Hackathon background

QingLu originated from the Meituan hackathon (Track 1) as a local-life AI wellness coach prototype.

The production codebase has moved to a self-hosted profile flow (`/onboard` → `/ready` → `/chat`) and no longer ships demo presentation scripts or fixed dialogue recordings.

For current setup and contribution guidelines, see the root [README](../README.md) and [CONTRIBUTING.md](../CONTRIBUTING.md).
