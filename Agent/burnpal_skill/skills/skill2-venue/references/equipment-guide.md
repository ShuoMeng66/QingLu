# 训练部位与器械对照表

> 用途：Skill 2 场景 E（日常场地推荐）的核心参考。用户说"今天练背/练腿/做有氧"，管家通过这张表将训练内容转化为器械需求，再按器械配置筛选健身房。
> 说明：本表覆盖主流商业健身房常见器械，不含小众或专业竞技器械。"必须有"指完成该部位训练的最低器械要求，"加分项"指能提供更丰富训练选择的额外器械。


## 背部训练

必须有：高位下拉机、坐姿划船机、引体向上架（至少有其中两项）
加分项：T 杠划船、龙门架（绳索）、单臂哑铃划船空间、反向飞鸟机
最低要求：有一组可调节哑铃 + 一个平凳即可完成基础背部训练（俯身划船）

## 胸部训练

必须有：卧推架（平板）+ 杠铃、可调节哑铃
加分项：上斜/下斜卧推架、龙门架（夹胸）、蝴蝶机、史密斯机
最低要求：一组哑铃 + 一个可调节角度的凳子

## 肩部训练

必须有：可调节哑铃、坐姿推举空间
加分项：史密斯机、龙门架（侧平举/前平举）、肩推机、哑铃架（多重量可选）
最低要求：一组哑铃即可完成推举、侧平举、前平举

## 手臂训练（二头 + 三头）

必须有：可调节哑铃、EZ 曲杠
加分项：龙门架（绳索弯举/下压）、牧师凳、绳索三头下压
最低要求：一组哑铃即可

## 腿部训练

必须有：深蹲架（含安全杠）+ 杠铃
加分项：腿举机、哈克深蹲机、腿弯举机、腿屈伸机、史密斯机
最低要求：深蹲架是腿部训练核心，没有深蹲架的健身房对力量训练很不友好
注意：深蹲架数量很关键。高峰期只有 1-2 个深蹲架的健身房会排队严重

## 核心训练

必须有：瑜伽垫/训练垫、平凳
加分项：悬垂举腿架、腹肌轮、龙门架（绳索卷腹）、瑞士球、药球
最低要求：一块垫子即可做平板支撑、卷腹等基础核心训练

## 有氧训练（室内）

必须有：跑步机（至少 3-5 台，避免排队）
加分项：椭圆机、划船机、动感单车、登山机/楼梯机
最低要求：跑步机。如果只做有氧不做力量，对器械区要求不高

## 有氧训练（户外）

适合场地：公园跑道、城市绿道、滨河步道、操场跑道
场地要求：路面平整、有照明（夜跑）、有标距（方便计算配速）
加分项：有直饮水点、有洗手间、环形路线（不用折返）

## 综合/功能性训练

必须有：功能性训练区（开阔空间）、壶铃、战绳、药球
加分项：TRX 悬挂训练带、跳箱、敏捷梯、弹力带
最低要求：一块开阔空间 + 基础自重即可

## 游泳

场地类型：25 米短池 / 50 米标准池
选择要点：水质（清澈无异味）> 泳道数量 > 距离
加分项：恒温泳池、独立更衣淋浴、有泳道分区（快速/慢速）
注意：部分健身房泳池较小（15-20 米），不适合认真游泳训练，只适合戏水放松


## 场地匹配速查

| 用户说的 | 翻译成器械需求 | 推荐场地类型 |
|---------|-------------|------------|
| "今天练背" | 高位下拉 + 划船机 + 引体架 | 器械齐全的商业健身房 |
| "今天练腿" | 深蹲架 + 腿举机 | 有自由力量区的健身房（注意深蹲架数量） |
| "今天练胸" | 卧推架 + 哑铃 | 有自由力量区的健身房 |
| "想做有氧" | 跑步机/椭圆机 或 户外跑道 | 任意健身房 或 公园跑道 |
| "想游泳" | 泳池 | 带泳池的健身房 或 专业游泳馆 |
| "想打球" | 球场 | 体育馆/球馆 |
| "想上团课" | 团操教室 | 有团课的连锁健身房 |
| "随便练练" | 基础器械即可 | 任意健身房，就近优先 |
