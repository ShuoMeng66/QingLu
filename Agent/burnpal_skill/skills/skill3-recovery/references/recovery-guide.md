# 运动后恢复评估指南

> 用途：Skill 3 场景 D（恢复评估）的判断依据。用户问"今天练完需要做恢复吗"，管家根据本指南评估是否需要专业恢复服务，还是自行恢复即可。
> 说明：本指南用于管家内部判断，不直接展示给用户。判断原则是保守但不过度——不把所有训练后都推付费服务，但该建议专业恢复时要明确建议。


## 评估维度

管家需要综合以下信息来判断：
1. 训练类型（力量/有氧/混合/球类）
2. 训练强度（自述或根据训练内容推断）
3. 训练时长
4. 训练频率（是否连续多天高强度）
5. 用户自述感受（酸痛程度、具体部位）


## 建议专业恢复的情况

以下情况建议用户做一次专业恢复服务（推拿/拉伸/按摩）：

1. **高强度力量训练超过 1.5 小时**
   - 大重量复合动作为主（深蹲、硬拉、卧推）
   - 多关节参与、大肌群训练
   - 训练后明显肌肉紧绷

2. **连续 3 天以上训练无休息日**
   - 累积疲劳导致恢复能力下降
   - 酸痛叠加，影响训练质量

3. **多日累积酸痛不消退**
   - 正常延迟性酸痛（DOMS）通常 48-72 小时消退
   - 超过 72 小时仍明显酸痛，说明恢复不充分

4. **大强度腿部训练后**
   - 腿部肌群体积大、恢复慢
   - 深蹲/腿举/硬拉等动作对下肢压力大
   - 深层放松能显著加速恢复

5. **久坐导致的持续肩颈不适**
   - 不是训练后的急性酸痛，而是长期姿势问题
   - 专业按摩或拉伸可以缓解并改善


## 自行恢复即可的情况

以下情况用户回家自己处理就够了，不需要推付费服务：

1. **中低强度训练，时长 1 小时以内**
   - 常规有氧（跑步 30-45 分钟、游泳、骑行）
   - 中等重量的力量训练
   - 团课（瑜伽/普拉提/搏击操）

2. **轻度酸痛，不影响日常活动**
   - 能正常走路、上下楼
   - 酸痛感在承受范围内

3. **规律训练者的日常酸痛**
   - 身体已适应训练强度
   - 这是正常的训练反应

### 自行恢复的方向性建议（管家可以简要提及）

- 训练后拉伸 10-15 分钟（不需要教具体动作，点到为止）
- 热水澡或泡澡 15-20 分钟
- 保证充足睡眠（7-8 小时）
- 补充蛋白质和水分
- 第二天可以做轻度活动（散步、轻度拉伸），促进血液循环


## 各训练类型的恢复重点

| 训练类型 | 主要疲劳部位 | 推荐恢复方式 | 适合的服务类型 |
|---------|------------|------------|-------------|
| 力量训练-上肢（胸/背/肩） | 胸大肌、背阔肌、三角肌、斜方肌 | 上背部和肩颈的深层放松 | 运动推拿、筋膜放松 |
| 力量训练-下肢（腿/臀） | 股四头肌、腘绳肌、臀大肌、小腿 | 下肢深层推拿 + 拉伸 | 运动推拿、被动拉伸 |
| 跑步/有氧 | 小腿、足底、髋屈肌 | 下肢拉伸 + 足底放松 | 拉伸服务、足底按摩 |
| 游泳 | 肩部、上背、核心 | 肩背放松 | 运动按摩、拉伸 |
| 球类运动 | 全身，尤其下肢和核心 | 全身放松 | 泰式按摩、运动推拿 |
| 久坐/办公 | 肩颈、腰背、髋屈肌 | 肩颈专项 + 腰背放松 | 肩颈推拿、拉伸 |


## 恢复服务类型简介

| 类型 | 特点 | 适合场景 | 力度 |
|------|------|---------|------|
| 中式推拿 | 按压穴位和经络，手法深沉 | 力量训练后的深层肌肉放松 | 强 |
| 运动按摩 | 针对运动后的专业手法，结合拉伸 | 高强度训练后、赛后恢复 | 中-强 |
| 被动拉伸 | 康复师辅助完成拉伸动作 | 柔韧性差、训练后放松 | 轻-中 |
| 泰式按摩 | 全身按压和拉伸结合，力度中等 | 全身放松、压力释放 | 中 |
| 日式指压 | 手指按压特定穴位，力度精准 | 局部酸痛、肩颈问题 | 中 |
| 筋膜放松 | 使用筋膜枪/筋膜刀等工具 | 筋膜粘连、长期紧绷 | 中-强 |
| 冰浴/冷疗 | 冷水浸泡，减少炎症和肿胀 | 高强度训练后、急性肌肉肿胀 | — |
| 热疗/汗蒸 | 促进血液循环和代谢废物排出 | 慢性疲劳、放松身心 | — |
