# 酸痛 vs 疼痛判断参考

> 用途：Skill 3 场景 X（就医引导）的核心判断依据。当用户描述身体不适时，管家需要区分"正常训练后酸痛"（可推荐恢复服务）和"可能受伤的疼痛"（应引导就医，不推荐任何恢复服务）。
> 原则：宁可保守判断。有疑虑就建议就医，不要为了推荐服务而忽视风险。误判为酸痛推荐了按摩，可能加重伤势；误判为疼痛建议就医，最多浪费一次门诊。两害相权取其轻。


## 正常酸痛的特征（可推荐恢复服务）

以下特征同时满足多条时，大概率是正常的延迟性肌肉酸痛（DOMS）：

1. **时间线**：训练后 12-72 小时出现，逐渐加重后逐渐减轻
2. **感受类型**：酸胀、紧绷、发硬、按压时酸痛，类似"使不上劲"
3. **范围**：整块肌肉区域的弥漫性酸痛（而非精确的某个点）
4. **对称性**：双侧都有（做了双侧训练的话），程度可能不完全相同
5. **活动影响**：活动受限轻微，热身后通常会缓解
6. **静止时**：不动时不痛或很轻微
7. **触感**：肌肉摸起来发硬/紧绷，但没有肿胀

用户可能的表述：
- "练完腿酸""背好紧""胳膊抬不太起来""上楼梯腿软"
- "浑身酸""使不上劲""感觉肌肉发胀"


## 异常疼痛的特征（应引导就医，不推荐恢复服务）

以下特征出现任意一条，就应该建议用户先就医排查：

1. **锐痛/刺痛**：明确的尖锐疼痛感，不同于酸胀的钝痛
2. **即时发生**：训练中或训练后立即出现的突然疼痛（正常 DOMS 不会立即出现）
3. **点状疼痛**：能精确指出"就是这个点疼"（关节、肌腱附着点等）
4. **肿胀**：疼痛部位明显肿起、发热
5. **淤青**：出现不明原因的淤血
6. **活动受限严重**：完全无法做某个动作（如完全不能弯膝盖、完全不能转头）
7. **弹响伴疼痛**：关节或肌腱处发出弹响声，同时伴随疼痛
8. **单侧疼痛**：做了双侧训练但只有一侧疼（提示可能不是训练导致的正常反应）
9. **静止时也痛**：不动也持续疼痛，甚至影响睡眠
10. **持续加重**：超过 72 小时不减轻反而加重

用户可能的表述：
- "膝盖刺痛""肩膀一动就疼""脚踝扭了肿了"
- "腰一弯就痛得不行""感觉哪里拉伤了""有个地方一按就很痛"
- "弹了一下然后就痛了""越来越痛了"


## 常见运动损伤信号

以下是常见的运动损伤类型和典型信号，管家不做诊断，只用于识别"应该就医"的信号：

| 损伤类型 | 典型信号 | 常见触发场景 |
|---------|---------|------------|
| 膝关节损伤（半月板/韧带） | 膝盖内侧或外侧锐痛、打软腿、上下楼梯痛、关节肿胀 | 深蹲时膝盖内扣、跑步突然变向、跳跃落地 |
| 肩袖损伤 | 抬臂到某个角度时锐痛、夜间痛、力量突然下降 | 过头推举、卧推过重、引体向上 |
| 腰椎间盘问题 | 腰部剧痛并向臀部/腿部放射、弯腰困难、坐久加重 | 硬拉弓背、久坐后突然发力 |
| 踝关节扭伤 | 脚踝外侧肿胀疼痛、不能负重行走 | 跑步踩不平路面、打球落地不稳 |
| 肌肉拉伤 | 突然的刺痛感、触摸有凹陷或硬结、淤青 | 未热身直接高强度运动、爆发力动作 |
| 肌腱炎 | 特定动作时的持续性疼痛、休息后僵硬活动后缓解 | 跑步膝（髂胫束）、网球肘、跟腱炎 |


## 管家的应对策略

### 判断为正常酸痛 → 可推荐恢复服务
- 正常回应，根据训练部位推荐对应的恢复服务
- 可以说"这是正常的训练反应，做个放松会恢复更快"

### 判断为可能受伤 → 引导就医
- 不推荐任何恢复服务（按摩/推拿/正骨等）
- 温和建议："这个情况建议先休息观察，如果不减轻建议去运动医学科看看"
- 如果用户坚持要推荐服务："理解你想快点好，但在确认是什么问题之前，按摩可能帮倒忙。看完医生说可以做被动放松了，再来找我推荐。"
- 急性损伤（扭伤/肿胀）可以简要提醒 RICE 原则：休息（Rest）、冰敷（Ice）、加压包扎（Compression）、抬高患肢（Elevation）

### 无法判断 → 保守处理
- 不推荐恢复服务
- "不太确定你的情况，安全起见建议先让医生看看。如果确认没事了再来找我推荐放松服务。"

### 绝对不能做的
- 不能说"可能是 XX 损伤"（管家不是医生，不做诊断）
- 不能说"做个推拿应该就好了"（可能延误治疗）
- 不能说"没事，练多了都这样"（可能忽视真正的损伤）
- 不能推荐"正骨"来处理疑似骨骼/关节问题（需要影像学检查确认后才能决定治疗方案）
